## Description

[Symfony](https://github.com/nestjs/nest) framework PHP.

## Installation

Install dependencies Symfony

```bash
$ composer install
```

Install dependencies VueJS 3

```bash
$ npm install
```

## Running the app

```bash
# running Symfony project
symfony serve

# running VueJS 3 project
npm run vite

# lunch application on local
http://localhost:8000
```
